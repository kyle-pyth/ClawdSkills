## Installation

### Via ClawHub (Recommended)

```bash
clawhub install <skill-name>
```

### Via Git

```bash
# Clone specific skill to your workspace
git clone https://github.com/kyle-pyth/ClawdSkills.git /tmp/clawdskills
cp -r /tmp/clawdskills/<skill-name> ~/.clawdbot/skills/
```

### Manual

Download the skill folder and copy it to your `skills/` directory.
